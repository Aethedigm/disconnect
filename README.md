# disconnect
Ebitengine Game Jame 2026

## Game Flow
On Start, game will launch into Main Menu. From Main Menu the options available will be Play (Go to Sortie Select), Options (Go to Options Menu), Quit (Exit Game on Desktop Only)  

Sortie Select will list missions available, after picking a mission will allow you to customize your Mecha with a top/bottom selector
